Transit Times

Average transit time per state from your warehouse (Kalyan, MH). Used to compute "days left to dispatch" on the Plan tab. Falls back to built-in defaults if not configured.

days
Set this to add a safety margin against optimistic courier estimates. Buffer is added to all lookup results (pincode-specific, state-wise, and default).
Example: pincode says 3d, buffer = 1 → TrackLens uses 4d. Set 0 for raw values.
Accepts state codes (MH, GJ) or full names (Maharashtra, Gujarat). Comments with # ignored.

Pincode-wise transit (preferred)

More accurate than state averages. When a pincode has data here, it overrides the state-wise lookup. Same table is used for all couriers.

Accepts 6-digit pincodes. Comments with # ignored. Large files OK — paste up to 20,000 rows.

Pincode lookup checker

Type a pincode (or pincode + state) to see which transit time TrackLens will use and where it came from. Useful for verifying coverage.

Honey Touch shipment dispatch & SLA tracker. Reads orders from InvoLens, fetches shipping labels from courier APIs, monitors status, and triggers alerts.
